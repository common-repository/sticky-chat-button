Display the beautiful Sticky Chat Button which can be customize to show on the WordPress frontend.

== Description ==

Sticky Chat Button is a Free & Minimal WordPress plugin that displays the beautiful Sticky Chat Button on the WordPress frontend.
After clicking on the Sticky Chat Button, It will bring you to the configured application or chat page.

**Admin Setting Panel**

* Sticky Chat Button - Settings
1. Activate/Deactivate Sticky Chat Button
2. Change the Position of Sticky Chat Button to Top-Left/Top-Right/Bottom-Left/Bottom-Right
4. Add Link to Open (Whatsapp, Tidio Chat, Talkto Chat
6. Customize Icon Width
7. Customize Icon Height
8. Customize Icon Margin
9. Change Icon Type to Default WhatsApp Icon/Custom Icon Image
10. Select Custom Icon
11. Save & Enjoy!!!

== Installation ==

1. Just log into your WordPress admin dashboard
2. Visit Dashboard > Sidebar > Plugins > Add New
3. Search for “Sticky Button – Click to Chat”
4. Install and Activate the “Sticky Button – Click to Chat” plugin
6. Activate the Sticky Button – Click to Chat to display on frontend (Mandatory)
7. Add a valid link to work with Sticky Button – Click to Chat
8. Set “Display on” functionality for the Mobile, Tablet & Desktop devices
9. Customize the Position, Width, Height, Margin, Type and Image of Icon (Optional)
10. Use & Enjoy!!!
